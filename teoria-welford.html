Dimostrazione di Welford

L'algoritmo di Welford serve a calcolare media e varianza in modo incrementale, cioè aggiornando i valori ogni volta che arriva una nuova osservazione, senza dover ricalcolare tutto da zero.

Supponiamo di osservare una sequenza di dati \(x_1, x_2, \dots, x_n\). Indichiamo con \(\mu_n\) la media dei primi \(n\) valori e con \(M_{2,n}\) la somma degli scarti quadratici dalla media:

\[ \mu_n = \frac{1}{n}\sum_{i=1}^{n} x_i \]
\[ M_{2,n} = \sum_{i=1}^{n} (x_i - \mu_n)^2 \]

L'idea chiave è trovare una formula ricorsiva che permetta di passare da \((\mu_{n-1}, M_{2,n-1})\) a \((\mu_n, M_{2,n})\) dopo l'arrivo del nuovo dato \(x_n\).

Aggiornamento della media

Partiamo dalla definizione:

\[ \mu_n = \frac{1}{n}\sum_{i=1}^{n} x_i = \frac{1}{n}\left(\sum_{i=1}^{n-1} x_i + x_n\right) \]

Poiché \(\sum_{i=1}^{n-1} x_i = (n-1)\mu_{n-1}\), otteniamo:

\[ \mu_n = \frac{(n-1)\mu_{n-1} + x_n}{n} \]

Riscrivendo la formula in modo più utile:

\[ \mu_n = \mu_{n-1} + \frac{x_n - \mu_{n-1}}{n} \]

Se definiamo \(\delta_n = x_n - \mu_{n-1}\), allora:

\[ \mu_n = \mu_{n-1} + \frac{\delta_n}{n} \]

Aggiornamento della quantità quadratica

Ora vogliamo aggiornare \(M_{2,n} = \sum_{i=1}^{n}(x_i - \mu_n)^2\). Separiamo l'ultimo termine:

\[ M_{2,n} = \sum_{i=1}^{n-1}(x_i - \mu_n)^2 + (x_n - \mu_n)^2 \]

Per i primi \(n-1\) termini scriviamo:

\[ x_i - \mu_n = (x_i - \mu_{n-1}) + (\mu_{n-1} - \mu_n) \]

Quindi:

\[ \sum_{i=1}^{n-1}(x_i - \mu_n)^2 = \sum_{i=1}^{n-1} \Big[(x_i - \mu_{n-1}) + (\mu_{n-1} - \mu_n)\Big]^2 \]

Sviluppando il quadrato:

\[ \sum_{i=1}^{n-1}(x_i - \mu_n)^2 = \sum_{i=1}^{n-1}(x_i - \mu_{n-1})^2 + 2(\mu_{n-1} - \mu_n)\sum_{i=1}^{n-1}(x_i - \mu_{n-1}) + (n-1)(\mu_{n-1} - \mu_n)^2 \]

Ma per definizione di media vale la regola che la somma degli scarti dalla media è zero:

\[ \sum_{i=1}^{n-1}(x_i - \mu_{n-1}) = 0 \]

Quindi il termine centrale sparisce e resta:

\[ \sum_{i=1}^{n-1}(x_i - \mu_n)^2 = M_{2,n-1} + (n-1)(\mu_{n-1} - \mu_n)^2 \]

Dato che \(\mu_n = \mu_{n-1} + \frac{\delta_n}{n}\), abbiamo:

\[ \mu_{n-1} - \mu_n = -\frac{\delta_n}{n} \]

Inoltre:

\[ x_n - \mu_n = x_n - \left(\mu_{n-1} + \frac{\delta_n}{n}\right) = \delta_n - \frac{\delta_n}{n} = \delta_n\left(1 - \frac{1}{n}\right) = \delta_n\frac{n-1}{n} \]

Sostituendo tutto in \(M_{2,n}\):

\[ M_{2,n} = M_{2,n-1} + (n-1)\left(\frac{\delta_n}{n}\right)^2 + \left(\delta_n\frac{n-1}{n}\right)^2 \]

Raccogliendo i termini:

\[ M_{2,n} = M_{2,n-1} + \delta_n^2 \frac{n-1}{n} \]

Ma poiché \(x_n - \mu_n = \delta_n \frac{n-1}{n}\), possiamo scrivere la formula finale nella forma classica di Welford:

\[ M_{2,n} = M_{2,n-1} + (x_n - \mu_{n-1})(x_n - \mu_n) \]

Varianza finale

Una volta aggiornato \(M_{2,n}\), la varianza si ottiene calcolando i seguenti rapporti:

Varianza della popolazione:

\[ \sigma_n^2 = \frac{M_{2,n}}{n} \]

Varianza campionaria:

\[ s_n^2 = \frac{M_{2,n}}{n-1} \]

Proprietà della media usate

  • Somma dei primi \(n-1\) termini:
    \[ \sum_{i=1}^{n-1} x_i = (n-1)\mu_{n-1} \]
  • La somma degli scarti dalla media è zero:
    \[ \sum_{i=1}^{n-1}(x_i - \mu_{n-1}) = 0 \]
